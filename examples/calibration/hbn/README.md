# HBN detector calibration example

The compressed calibrant and dark images are supplied measurement inputs. The compact NPZ retains
only fit points, ellipse parameters, center and tilt values, confidence, and residual summaries.
It intentionally omits 3000 by 3000 derived arrays and absolute legacy paths.

This reference is useful for the later detector-calibration task, but analytic and synthetic
recovery remain the authority for coordinate and tilt conventions.
